## ShortVideo

### 说明

#### 基于react-native开发的类抖音App

### 下载并运行


```bash
# Clone this repository
$ git clone https://github.com/hizouxx/ShortVideo.git

# Go into the repository
$ cd RN-Girls

# Install dependencies
$ yarn install

# Run the app (Android)
$ yarn android

# Run the app (IOS)
$ yarn ios

```
