## 功能
### 已完成功能

 - 短视频上滑下滑展示
 - 评论信息展示
 - 个人信息展示
 - 点赞（Mock数据，并非真正更改点赞状态）
### 待完成功能

 - 转发功能
 - 发布短视频功能
 - 发布评论功能

## 下载并运行

```bash

# Clone this repository
$ git clone https://github.com/hizouxx/ShortVideo.git

# Go into the repository
$ cd ShortVideo

# Install dependencies
$ yarn install

# Run the app (Android)
$ yarn android

# Run the app (IOS)
$ yarn ios

```
